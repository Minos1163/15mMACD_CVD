AI增强型虚拟货币网格交易系统

### 1.4 文档目标

本文档用于统一定义系统中的配置参数，包括：

* 参数名称
* 字段名
* 类型
* 默认值
* 取值范围
* 是否必填
* 生效范围
* 参数说明
* 风控限制
* 前后端展示与接口约束

---

## 2. 使用说明

### 2.1 参数分类

建议将参数分为以下几类：

* 基础参数
* 交易参数
* 网格参数
* AI参数
* 风控参数
* 执行参数
* 监控与日志参数
* 系统参数

### 2.2 参数优先级

建议约定以下优先级：

1. 交易所约束
2. 风控硬限制
3. 用户显式配置
4. AI推荐值
5. 系统默认值

### 2.3 命名规范

建议统一使用：

* 配置字段：`snake_case`
* 枚举值：小写英文
* 布尔值：`true / false`
* 百分比统一使用小数表示，如 `0.05 = 5%`

# 3. 参数配置总表

## 3.1 基础参数

### 3.1.1 `symbol`

* **中文名称**：交易对
* **字段名**：`symbol`
* **类型**：string
* **必填**：是
* **默认值**：`DOGEUSDT`
* **示例值**：`DOGEUSDT`
* **说明**：策略运行的目标交易对
* **生效范围**：策略级
* **约束**：必须为交易所支持的有效交易对
* **备注**：MVP阶段固定为 `DOGEUSDT`

---

### 3.1.2 `capital`

* **中文名称**：投入本金
* **字段名**：`capital`
* **类型**：number
* **必填**：是
* **默认值**：`100`
* **单位**：USDT
* **最小值**：根据交易所最小下单额与策略要求设定
* **说明**：策略初始投入资金
* **生效范围**：策略级
* **约束**：必须大于最小策略启动资金
* **备注**：MVP默认 `100 USDT`

---

### 3.1.3 `account_mode`

* **中文名称**：账户模式
* **字段名**：`account_mode`
* **类型**：string
* **必填**：否
* **默认值**：`isolated`
* **可选值**：

  * `isolated`
  * `cross`
* **说明**：保证金模式
* **生效范围**：账户级 / 策略级
* **约束**：需与交易所支持模式一致

---

# 5. 参数校验规则模板

## 5.1 通用校验规则

* `price_lower < price_upper`
* `grid_count > 0`
* `active_order_count <= grid_count`
* `leverage <= max_leverage`
* `0 <= max_margin_usage <= 1`
* `0 <= max_drawdown_threshold <= 1`
* `0 <= daily_loss_limit <= 1`
* `0 <= ai_confidence_threshold <= 1`
* `0 <= risk_score_threshold <= 100`

## 5.2 联动校验规则

* 启用 AI 自动切换时，必须同时启用 `ai_enabled`
* 若 `moving_grid_enabled = true`，建议限制较低杠杆
* 若 `market_type = futures`，则必须校验 `leverage`
* 若 `dry_run = true`，应禁止真实交易账户下单
* 若 `grid_mode = short`，需校验交易所是否支持对应空头机制
