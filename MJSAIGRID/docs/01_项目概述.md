# 项目概述

## 1.1 项目名称

AI增强型加密货币网格交易系统（DOGEUSDT 首发版）

## 1.2 项目目标

- 基于 `DOGEUSDT` 实现可运行的网格交易系统
- 初始资金 100 USDT，小资金验证策略有效性
- 支持中性 / 做多 / 做空网格动态切换
- 支持 2X–20X 杠杆配置
- 引入 AI 模块辅助判断市场状态并动态调整参数
- 提供回测、模拟盘、实盘三阶段部署能力

## 1.3 项目边界

### 本期包含

- 单交易对：DOGEUSDT
- 单账户运行
- 网格策略执行引擎
- AI信号模块
- 风控模块
- 回测与监控面板

### 本期不包含

- 多交易对组合管理
- 跨交易所套利
- 高频撮合优化
- 链上数据因子系统
- 复杂组合对冲系统

## 1.4 业务价值

- 降低人工盯盘成本
- 在震荡行情中获取网格收益
- 在趋势行情中通过 AI 避免"逆势死扛"
- 建立可扩展的量化交易底座
