# 网格交易概念与术语

## 3.1 网格交易原理

### 3.1.1 基本原理

网格交易是一种在设定价格区间内分布买卖挂单的策略。价格下跌时分批买入，价格上涨时分批卖出，通过波动反复赚取价差。

### 3.1.2 核心特征

- 自动化执行：预设价格区间和网格层数，系统自动挂单
- 风险可控：每格止损有限，整体风险可控
- 适合震荡：在区间震荡行情中收益稳定
- 反脆弱：在一定范围内，价格波动越大收益越高

## 3.2 核心术语

### 3.2.1 价格区间

| 术语 | 说明 |
|-----|------|
| price_lower | 网格区间下限 |
| price_upper | 网格区间上限 |
| grid_count | 网格总层数 |
| active_order_count | 活跃挂单数量 |

### 3.2.2 网格类型

| 类型 | 说明 | 适用场景 |
|-----|------|---------|
| arithmetic | 等差网格，每格价格差相同 | 价格绝对波动稳定 |
| geometric | 等比网格，每格价格比例相同 | 高波动资产如DOGE |

### 3.2.3 网格模式

| 模式 | 说明 | 适用行情 |
|-----|------|---------|
| neutral | 中性网格，双向挂单 | 震荡市场 |
| long | 做多网格，偏多头持仓 | 上涨趋势 |
| short | 做空网格，偏空头持仓 | 下跌趋势 |

### 3.2.4 杠杆术语

| 术语 | 说明 |
|-----|------|
| leverage | 杠杆倍数 |
| margin | 保证金 |
| position_size | 持仓数量 |

### 3.2.5 风控术语

| 术语 | 说明 |
|-----|------|
| stop_loss_price | 止损价 |
| take_profit_price | 止盈价 |
| max_drawdown | 最大回撤 |
| risk_score | 风险评分 |

## 3.3 等比网格计算公式

### 3.3.1 网格价位计算

在区间 [L, U] 内，若划分 N 格，则比例因子为：

```
ratio = (U / L)^(1 / N)
```

第 i 层价格为：

```
price_i = L * ratio^i
```

### 3.3.2 单格收益

单格理论收益 = 卖价 - 买价 - 手续费 - 滑点

### 3.3.3 活跃挂单机制

- 只维护距离当前价格最近的若干层订单
- 成交后自动在相邻网格层补单
- 降低保证金占用，提高资金效率
