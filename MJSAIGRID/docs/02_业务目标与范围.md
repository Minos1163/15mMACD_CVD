# 业务目标与范围

## 2.1 业务目标

### 2.1.1 核心目标

构建一套针对 DOGEUSDT 的 AI增强型网格交易策略，在控制回撤和爆仓风险的前提下，通过市场波动获取稳定收益。

### 2.1.2 具体目标

- 在震荡行情下，通过中性网格实现高频价差收益
- 在明显趋势行情下，通过 AI 辅助切换至做多或做空网格
- 控制高杠杆带来的风险，确保小资金账户可持续运行
- 支持回测、模拟盘和实盘统一策略逻辑
- 提供可解释的 AI 决策和可审计的执行日志

### 2.1.3 设计原则

- 风控优先于收益
- 先稳定，后优化
- 先规则AI，后复杂模型
- 先小额验证，后扩大规模
- 策略逻辑可解释、可复盘、可调优

## 2.2 项目范围

### 2.2.1 功能范围

| 功能模块 | 优先级 | 说明 |
|---------|--------|------|
| 网格引擎 | P0 | 核心网格生成与执行 |
| 订单管理 | P0 | 订单下单、撤单、同步 |
| 模式切换 | P1 | 中性/做多/做空切换 |
| AI趋势分析 | P1 | 市场状态分类与建议 |
| 风险管理 | P0 | 杠杆、止盈止损、熔断 |
| 参数配置 | P0 | 配置管理与参数校验 |
| 回测系统 | P2 | 历史数据回测验证 |
| 监控面板 | P2 | 运行状态监控与告警 |

### 2.2.2 技术边界

- 交易对：仅 DOGEUSDT
- 初始资金：100 USDT
- 杠杆范围：2X-20X
- 网格模式：中性/做多/做空
- 网格类型：等差/等比

### 2.2.3 不包含范围

- 多交易对组合管理
- 全自动AI策略切换(V1仅提供建议)
- 高杠杆激进模式
- 强化学习参数自适应
- 多因子复杂模型
